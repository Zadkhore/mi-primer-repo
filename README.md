# mi-primer-repo
Trabajo de EDA II
El código define tres funciones que se usan en cada lista

double(arr): Toma una lista y devuelve una nueva lista con cada número multiplicado por 2

maximum(arr): Busca y devuelve el valor más grande de la lista, pero si esta vacia devuelve None

average(arr): Calcula el promedio de la lista cuma todo y lo divide entre la cantidad de elementos, si la lista está vacía regresa un 0, para evitar problemas al dividir

La función main ejecuta 4 diferentes listas para probar el codigo:

Se tienen cuatro listas diferentes:

lista1: [1, 2, 3, 4, 5]

lista2: [10, 20, 5, 40]

lista3: [-5, 0, 5, 10]

lista4: [0, 0, 0, 0]

Imprime en la consola cada lista y aplica las tres funciones (double, maximum, average) a cada lista e imprime los resultados
